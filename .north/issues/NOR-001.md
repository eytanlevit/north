**Context**: Enable users to paste screenshots directly into the chat interface, allowing richer visual context when conversing with the AI agent, matching Claude Code's functionality.

**Requirements**:
- Support pasting images from clipboard (Cmd/Ctrl+V) - screenshots primarily
- Support image formats: png, jpg, gif
- Store attachments temporarily in /tmp/ (session-only, not persistent)
- Display attachments as filename/icon in chat (e.g., 📎 screenshot.png)
- Send attachments to AI model as multimodal content via Anthropic API

**Acceptance Criteria**:
- User can paste a screenshot (Cmd/Ctrl+V) and it appears in chat as filename
- Image is encoded and sent to the API (base64)
- AI receives and can reference the image in responses

**Technical Details**:
- Files: `src/components/chat-pane.ts` (handle clipboard events)
- Detect clipboard paste with image data
- Save to /tmp/ with unique filename
- Update message format to support multimodal content (text + images)
- Encode images as base64 for Anthropic API
- Display as simple filename with icon in chat history
- No need for terminal image preview protocols (keep simple)
