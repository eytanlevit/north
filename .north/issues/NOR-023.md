## Problem
When switching between panes using Tab, there's insufficient visual indication of which pane is currently focused.

## Expected Behavior

### Agent Pane
- **When focused**: Cursor should be visible
- **When unfocused**: Cursor should disappear

### Issues Pane (Kanban Board)
- **When focused**: Issue highlighting should be visible
- **When unfocused** (moving to project pane): Issue highlighting should disappear

## Current State
The visual feedback doesn't clearly indicate focus state, making it unclear which pane is currently active.

## Acceptance Criteria
- [ ] Cursor disappears when leaving agent pane
- [ ] Cursor reappears when returning to agent pane
- [ ] Issue highlighting disappears when leaving issues pane
- [ ] Issue highlighting reappears when returning to issues pane
- [ ] Tab navigation clearly shows which pane has focus
