**Context**: Enable dark mode throughout the TUI to reduce eye strain and provide a modern, customizable user experience.

**Requirements**:
- Implement dark color palette for all UI components (board, chat panel, dialogs, status bars)
- Add toggle mechanism (keyboard shortcut or command) to switch between light/dark modes
- Persist user's theme preference across sessions
- Ensure text readability and proper contrast in dark mode
- Apply dark mode to markdown rendering in issue bodies and comments

**Acceptance Criteria**:
- User can toggle dark mode via keyboard shortcut and preference is saved
- All UI elements are visible and readable in dark mode with proper contrast
- Theme preference persists after restarting the application
