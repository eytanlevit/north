**Context**: Provide visual feedback when the AI agent is processing or thinking, improving UX by showing users that work is happening and how long it's taking.

**Requirements**:
- Display animated Braille spinner (⠋⠙⠹⠸⠼⠴⠦⠧⠇⠏) inline in chat history (fixed style, not configurable)
- Show elapsed time always (e.g., "⠋ Thinking... (3s)")
- Show context-aware messages based on agent activity
- When multiple tools run in sequence, show list: "⠋ Running: read, create_issue (5s)"
- Appear immediately when user submits a message
- Remove completely when agent starts streaming response

**Acceptance Criteria**:
- Spinner appears immediately after user submits message with elapsed timer starting at 0s
- Message updates based on tool being used (context-aware)
- When multiple tools are active, shows list of tool names
- Spinner disappears cleanly when streaming text begins
- Animation is smooth without flicker

**Technical Details**:
- Files: `src/components/chat-pane.ts`, `src/index.ts` (session event handling)
- Hook into session events: `agent_start`, `tool_execution_start/end`, `message_update`, `agent_end`
- Add `setThinkingStatus(message: string | null)` method to ChatPane
- Rotate spinner chars every 80-100ms
- Track elapsed time from agent start, update display every second
- Maintain list of active tools, display as comma-separated
- Context-aware message mapping:
  - Default: "⠋ Thinking... (2s)"
  - Single tool: "⠋ Reading files... (3s)"
  - Multiple tools: "⠋ Running: read, grep, create_issue (8s)"

**Out of Scope**:
- Customizable spinner styles (stick with Braille)
- Progress bars or percentage indicators
- Per-tool timing breakdown
