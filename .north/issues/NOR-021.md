**Context**: Add timer functionality to help users track time spent on tasks and manage work sessions effectively.

**Requirements**:
- Implement timer that can be started/stopped/reset via keyboard shortcuts
- Display active timer in the UI (status bar or issue card)
- Persist timer state when switching between issues or closing the app
- Track and store time spent per issue
- Provide visual indication when timer is running

**Acceptance Criteria**:
- User can start/stop a timer and see elapsed time update in real-time
- Time tracking data persists across app restarts
- User can view total time spent on each issue
