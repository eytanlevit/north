Add a `/issue` command that signals clear intent to create an issue, triggering the agent to ask clarifying questions and create a well-formed issue.

## Purpose
While users can ask to create issues in natural language, `/issue` provides a clear, explicit command that says "I want to create an issue now, help me make it good."

## Behavior
1. User types: `/issue <description>`
2. Agent analyzes the input
3. If vague/unclear, agent asks clarifying questions using the questions tool
4. If detailed enough, agent proceeds directly
5. Agent creates a reasonably well-formed issue
6. Minimal confirmation message shown: "Created NOR-XXX"

## Examples

**Vague input (agent asks questions):**
```
User: /issue add timer
Agent: [Questions tool popup]
  - What should the timer track? (time spent on issues, pomodoro, etc.)
  - Where should it display? (UI, CLI, both?)
  - Any specific requirements?
Agent: Created NOR-016: Add time tracking timer to issue detail view
```

**Detailed input (agent creates directly):**
```
User: /issue add a timer feature that tracks time spent on issues and displays in the issue detail overlay
Agent: Created NOR-017: Add time tracking timer to issue detail view
```

## Difference from Natural Language
- **No real functional difference** - just clearer intent
- Agent knows user definitely wants to create an issue (no "should I create this?" confirmation)
- Consistent, predictable workflow

## Difference from /enrich (NOR-011)
- `/issue` - Creates new issues from scratch with basic clarification
- `/enrich` - Takes existing issues and deeply enriches them with research, code analysis, and comprehensive detail
- `/enrich` does online research, `/issue` does not

## Technical Implementation
- Parse `/issue` prefix in `chatPane.onSubmit`
- Extract description after `/issue`
- Send to agent with system context: "User wants to create an issue. Description: {input}. Ask clarifying questions if needed, then create a well-formed issue."
- Agent uses existing tools: `ask_questions`, `create_issue`
- Show minimal confirmation message in chat
