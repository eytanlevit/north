Prevent accidental exits by requiring double Ctrl+C press within 3 seconds to quit the application.

## Current Behavior
- Single Ctrl+C immediately exits the TUI
- Easy to accidentally quit

## Desired Behavior
- First Ctrl+C shows message: "Press Ctrl+C again to exit"
- Must press Ctrl+C again within 3 seconds to actually exit
- After 3 seconds, resets (next Ctrl+C shows message again)
- Message appears in chat pane or as status indicator

## Implementation
- Track timestamp of first Ctrl+C press
- Show temporary message on first press
- Check elapsed time on second press
- Only exit if within 3 second window
- Reset state after timeout

## Technical Considerations
- Update global key handler in `src/index.ts`
- Maintain state: `lastCtrlCTime` or similar
- Display confirmation message (where? chat pane? overlay? status bar?)
- Clear message after timeout or on second press
- Ensure timer doesn't leak or block

## UX Details
- Message should be non-intrusive but visible
- Could show countdown: "Press Ctrl+C again to exit (3s remaining)"
- Consider color (yellow/warning color?)
- Should not interrupt current workflow
